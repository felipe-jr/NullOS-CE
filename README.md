 Fase 0: Boot
What is this?
NullOS is a hobby operating system written from scratch in C and x86 Assembly. The twist: every single line of code was generated by AI (Claude). This project exists to answer one question: how far can AI go when building something as complex as an OS?

This is not meant to be a production OS. It's an experiment.

Building
Dependencies
# Fedora

sudo dnf install -y nasm xorriso qemu-system-x86 docker

# Start Docker

sudo systemctl start docker

sudo usermod -aG docker $USER
Build with Docker (recommended)
git clone https://github.com/SussyNade/NullOS.git

cd NullOS

docker run -it --rm -u root -v $(pwd):/nullos:z randomdude/gcc-cross-x86_64-elf

# Inside the container:

apt-get install -y nasm grub-pc-bin grub-common xorriso mtools -q

cd /nullos/tools && make

exit
Run in QEMU
qemu-system-x86_64 \

  -cdrom build/nullos.iso \

  -m 256M \

  -no-reboot \

  -no-shutdown \

  -display sdl
